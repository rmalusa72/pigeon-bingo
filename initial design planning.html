Pigeon Bingo

Pigeon Bingo is a game which teaches about pigeon genetics and encourages appreciation of maligned birds.

When a user sees a pigeon, they can log it in the app. Based on the pigeon’s genes, the user will get points. The app will store basic user data (what pigeons they have seen, their score, etc.) as well as information on pigeon genes and coloration.

Brainstorming User Stories

  1. When I open the app, I am logged in to a profile (associated with the device by default).
  2. Instead of loading the default account, I can use an identifier to load an existing profile. In either case, I become “logged in”.
  3. When I open the app after previously using it, it will load my existing profile and log me in by default.
  4. As a logged-in user, I can access an identifier that can be used to load my profile on other devices.
  5. As a logged-in user, I can enter an identifier to switch profiles.
  6. As a logged-in user, I can access a “pigeon interface” which allows me to log a new pigeon. The pigeon is then added to my “pigeon sightings” and I gain points.
  7. From the pigeon interface, I can follow a flowchart to identify and then log a pigeon I see.
  8. In the pigeon interface flowchart, there are illustrations as well as descriptions in words.
  9. From the pigeon interface, I can select genes to “create a pigeon” and then confirm that it matches what I am seeing, which logs it.
  10. Selecting genes to “create a pigeon” also generates a representative illustration.
  11. As a logged-in user, I can view my pigeon sightings. Each sighting has information on the pigeon in question.
  12. The sightings are also labeled with where and when I saw them.
  13. I can add a picture to a pigeon sighting.
  14. I can click on a pigeon sighting to see any pictures I have added to it.
  15. I can sort the pigeon sightings by point value or recency.
  16. The pigeon sightings are represented by illustrations as well as verbal descriptions.
  17. I can view the representative illustrations overlaid on a map.
  18. The pigeon sightings view also allows me to see which alleles of each gene I have spotted.
  19. When I log a pigeon, I earn points based on the rarity of its genes.
  20. When I have logged a pigeon with each allele of a gene, I earn bonus points.
  21. When I log a certain number of pigeons, or a certain number of pigeons of a particular phenotype, I earn bonus points.
  22. I can see these bonus point opportunities in the form of “achievements” which may be visible or hidden.
  23. I can delete a pigeon sighting if I made a mistake, and points and other data are updated accordingly.
  24. I can share a pigeon sighting on Twitter.
  25. I can create an avatar using the “create a pigeon” screen, with additional backgrounds and accessories.
  26. (Admin) I can use a web app to add a new gene with alleles.

Basic Database Structure

  • Users
  • Identifier  
  • Pigeon sightings
  • Time
  • Location
  • Pigeon
  • Genotype
  • Genes
  • Phenotype
  • Base point value
  • Current points
  • Achievements and gene statistics
